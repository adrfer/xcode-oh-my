# Xcode Oh My
Just a quirky Xcode profile, in all its glory.

This is an umbrella project where:

- Remarkably uncharming key bindings for Xcode live at [xcode-keybindings](https://github.com/adrfer/xcode-keybindings)
- Minimally ok-ish themes for Xcode live at [xcode-themes](https://github.com/adrfer/xcode-themes)
- A jazzy arsenal of Swift snippets for Xcode live at [xcode-snippets](https://github.com/adrfer/xcode-snippets)
